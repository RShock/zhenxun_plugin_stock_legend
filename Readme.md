# 股海风云 V3.0

适配真寻最新版（基于 Alconna 指令系统）的炒股小游戏，使用真寻经济系统金币，真实股市数据。

进行了严格的防风控处理，所有消息均为图片发送（Windows 下持仓同样为图片）。

### V3.0 更新内容

* **Alconna 指令系统**：从 `on_command` 全面迁移至 `on_alconna`，支持更灵活的指令匹配（如 `买股票601919 1000 5` 一行直出），参数顺序容错、自动纠错
* **委托单系统**：美股非交易时段创建委托单等到开盘执行，港股 T+0 但延迟 15 分钟成交，A 股即时成交。买入委托创建时立即扣款，定时任务每 60 秒检查执行
* **跨群账户合并**：默认开启，同一 QQ 在不同群共用一个股海风云账户，自动迁移旧 `QQ:群号` 格式的持仓和委托单数据
* **取消委托**：新增 `取消委托` 指令，可取消所有待执行委托单并返还已扣金额
* **多图源切换**：新增新浪财经、网易财经、腾讯财经 K 线图源，支持配置自动选择（依次尝试新浪→网易→腾讯），大幅提升加载速度和稳定性
* **数据库修复**：`stock_game.number/cost` 和 `stock_order.cost` 列类型从 `NUMERIC(10,3)` 扩展至 `NUMERIC(20,3)`，防止大额金币溢出，启动自动 ALTER
* **超时委托退款**：超时 3 小时以上的失败委托单自动退还已扣金币
* **不计其数的细节优化**：多处参数防呆、杠杆/金额顺序自动纠正、卖空限制完善、涨停板保护增强等

### 指令

`买股票 代码 金额 杠杆倍数(可不填)` 买入股票  
例：`买股票 600888 10000 5`（买入 10000 金币的 5 倍杠杆仓位）

多功能指令，包含以下逻辑：
* 只输入代码：默认满仓最高杠杆梭哈
* 不输入杠杆：默认最大倍率（如果之前买过则沿用之前杠杆）
* 杠杆 > 最大倍率但买入数量 < 杠杆：自动纠正参数顺序
* 买入数量 ≤ 10：视为"仓位"而非"金额"
* 金额为 0 杠杆不为 0：修改杠杆
* 金额为 0：仅查看股票行情
* 金额为负数且 ≥ -最大杠杆：满仓做空

`卖股票 代码 仓位(十分制)`  
例：`卖股票 600888 10`（卖出全部仓位）

`我的持仓` — 查看自己的持仓和委托单

`查看持仓@qq号` — 查看别人的持仓

`反转持仓 代码` — 短线快捷多转空/空转多

`清仓` — 卖出所有股票

`取消委托` — 取消所有待执行委托单并退款

`关于股海风云` — 查看版本和更新链接

`查看股票 代码` — 查看股票 K 线图

`强制清仓 qq号` — 管理指令，给爆仓的人平仓

### 支持范围
* A 股、港股、美股、基金、日股
* 做空、杠杆（最大倍率可配置）
* 委托单（港股/美股）
* 躺平基金（每日复利增长）

### 配置项

| 配置项 | 默认值 | 说明 |
|--------|--------|------|
| 最大杠杆比率 | 5 | 最大杠杆倍率 |
| 躺平基金每日收益 | 0.015 | 每日复利比率（1.5%） |
| WIN_FIT | false | Windows 兼容模式 |
| 跨群合并账户 | true | 同一 QQ 跨群共用账户 |
| 股票提示图模式 | 3 | 1:百度 2:东方财富 3:新浪(推荐) 4:网易 5:腾讯 6:自动 |
